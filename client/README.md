# DATTT Client

## Dependencies
This client uses:
- `next version 13.1.6`
- `react version 18.2.0`
- `@solana/web3.js version 1.73.2`

## Setup
```
yarn install
```

## Run
For development run
```
yarn dev
```
or
```
npm run dev
```
